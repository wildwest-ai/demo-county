# demo-county

> **This is a demo county instance** for the [Wild West AI governance framework](https://github.com/wildwest-ai/wildwest).
> It shows what a real county looks like when fully instantiated. Use it as a reference when setting up your own county.

---

## What This Is

A fictional county — `tombstone` — governed under the Wild West framework. Sheriff is `pat` (Pat Garrett). Town Marshal is `wyatt` (Wyatt Earp). One town: `tombstone-app`.

It is not a real project. Every actor, town, and rule here is illustrative.

---

## How To Use This

Read the files in order:

1. [`CLAUDE.md`](CLAUDE.md) — county law; shows the full actor model, standing rules, authority framework, and chat notation populated for a real county
2. [`registry.json`](registry.json) — author registry and town list
3. [`dev-pairs/README.md`](dev-pairs/README.md) — full dev-pairs roster

Then instantiate your own county following the [framework README](https://github.com/wildwest-ai/wildwest/blob/main/README.md#instantiating-a-county).
